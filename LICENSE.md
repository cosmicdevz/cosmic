**Enable auto-inject and switch to the cosmic theme via CLI patch:**

```bash

# Patch auto_inject to true using sed (Git Bash / WSL)
```

## 🚨 Troubleshooting & Common Issues

### ❌ "Injection Failed — Process Not Found"

**Cause:** Roblox wasn't fully loaded when you clicked Inject.
**Fix:** Increase `inject_delay_ms` in your config to `5000` or `6000`, or manually wait until your popular title's lobby screen is fully rendered before injecting.

```bash
sed -i 's/"inject_delay_ms": 3000/"inject_delay_ms": 6000/' config.json
```

---

### ❌ "DLL Load Error on Windows"

**Cause:** Windows Defender quarantined the DLL during or after installation.
**Fix:**
1. Open Windows Security → Virus & Threat Protection → Protection History
2. Find the quarantined file and select "Restore"
3. Add `C:\CosmicExec\` to your Defender exclusions list

---

### ❌ "Script Hub Shows Blank / Won't Load"

**Cause:** Hub cache is corrupted or the CDN is temporarily unreachable.
**Fix:**
```bash
# Clear the local hub cache
rm -rf %APPDATA%/CosmicExecutor/hub_cache/

# Re-launch executor — cache will rebuild automatically
```

---

### ❌ "iOS App Crashes Immediately on Launch"

**Cause:** Developer certificate expired (this happens every 7 days on free Apple IDs).
**Fix:** Open AltStore on your iPhone → Apps tab → tap "Refresh" next to cosmic executor. Takes ~15 seconds on WiFi.

---

### ❌ "Executor Crashes When Executing Large Scripts"

**Cause:** Default memory allocation is too conservative for heavy scripts.
**Fix:** Open config and increase the thread memory ceiling:
```bash
sed -i 's/"thread_mem_mb": 64/"thread_mem_mb": 128/' config.json
```

---

### ❌ "Android APK Install Blocked"

**Cause:** "Install Unknown Sources" is disabled.
**Fix:** Settings → Apps → Special App Access → Install Unknown Apps → select your file manager → Allow.
